# Place Canada 2026 (Tyles Edition)

Welcome to the **Place Canada 2026** coordination hub\! This repository contains the configuration and scripts needed to help us maintain our canvas presence on Tyles.

Special thanks to **@Vio (Neurosama)** for the software and template plugin support\!

---

## 🚀 How to Participate

Follow these steps to get the template overlay running on your browser:

### 1. Install Tampermonkey

You need a userscript manager to run the overlay software.

- [Download Tampermonkey for Chrome/Edge/Firefox](https://www.tampermonkey.net/)

### 2. Install the Overlay Script

We use a modified version of the Charity script to handle the template positioning.

- **[Click here to install the script](https://eliv.moe/charity)**

### 3. Launch Tyles with our Template

Once the script is installed, use the link below to open Tyles with the Canada template pre-loaded:

- **[Open Tyles (Canada Config)](https://tyles.place/?charity=https://raw.githubusercontent.com/NoSadBeHappy/canadaplace/refs/heads/main/placecanada.json)**
